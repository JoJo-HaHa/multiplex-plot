4. Graph¶

Graphs, sometimes referred-to as networks, show the structure of data through nodes and edges. Each graph is made up of nodes and edges and can be directed or undirected.

Keep reading to learn more about graph visualizations.

Note

Check out the graph tutorial Jupyter Notebook.

The Graph class can be used to draw directed or undirected graphs. Each graph is made up of nodes and edges and can be used to show the relations between nodes.

Note

The graph visualization uses the networkx package to generate the layout of the graph. Therefore it is a prerequisite to create these visualizations.
